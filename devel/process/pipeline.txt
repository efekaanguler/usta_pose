get the session folder path

under it we will have calib_data folder cam1 2 3 4 folders and frame timing metadata etc.

first we have to process frames. by matching them. select a master camera (select the one that we assume as the master in record stage)

starting from the 10th frame of the master camera, match the frames of other cameras to it for each frame (find a corresponding frame in other words)

at the end of that process we will have n number of matched frames. then for each frame for all cameras apply the following:

for pose cameras (cam1 and cam2):
apply rtmw2d (as a result we will obtaing 2d points for markpoints representing pose)
for each frame and each point find the corresponding depth value from the depth video of the same camera

for gaze cameras (cam3 and cam4):
apply puregaze (as a result we will obtain gaze vector)

write them to a new csv-> each mark from pose model as 3d + gaze vectors + timestamp and corresponding frame number and any relevant information
